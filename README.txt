INTRODUCTION
------------
The folder contains two things as follows:
 * data: contains all images used in our experiments.
 * src: contains our source code implemented in Matlab. 

REQUIREMENTS
------------
Matlab R2012b

HOW TO RUN
------------
Using Matlab R2012b
 * Open the file 'run.m' in the folder 'src' 
 * Click Run (F5)

NOTE: You can change the test images in the 'run.m'.

REFERENCE
------------
Rang M.H. Nguyen, Seon Joo Kim, Michael S. Brown,
Illuminant Aware Gamut-Based Color Transfer,
Computer Graphics Forum 2014.
